# Map with OSRM Routing

This project displays an interactive map using Leaflet and React-Leaflet with routing powered by OSRM (Open Source Routing Machine).

## Features

- Interactive map using OpenStreetMap tiles
- Route calculation between two points using OSRM
- Display of route distance and estimated travel time
- Visual route polyline on the map
- Start and end point markers

## Usage

The map displays a route from Delhi (start point) to another location in Delhi (end point). The route is automatically fetched from the OSRM API and displayed on the map.

You can modify the `startPoint` and `endPoint` coordinates in `MapWithOSRM.jsx` to display routes between different locations.

## OSRM API

This project uses the public OSRM demo server: `https://router.project-osrm.org`

For production use, consider hosting your own OSRM instance.
